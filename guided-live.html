Timeline

What changed from the beginning

The walkthrough below follows the actual progression of the website based on the work we shipped together.

Foundation

Brand cleanup and header refinement

We started by tightening branding consistency, fixing logo handling, and improving header spacing so the site felt more polished before larger experience changes landed.

Logo naming fix Header spacing COREHUB™ wordmark

Experience Layer

Hero sections and dashboard storytelling were strengthened

Once the brand foundation was in place, we aligned hero banners, refreshed dashboard presentation, removed the dashboard hero image behind the blue banner, and improved the way the site introduced the product story across major pages.

Hero alignment Dashboard framing Visual consistency

Conversion Flow

Get Started modals and signup entry points were standardized

We then unified how visitors entered the product from across the site so the calls to action felt consistent and the routing behavior became easier to reason about.

Shared modal pattern Clear trial CTA Cleaner navigation

Major Split

User signup and coaching-plan signup now serve different journeys

The biggest architectural improvement was separating the signup experiences. The dashboard path now uses a user signup flow for Individual User, Employer / HR Leader, and Small Business, while coaching products uses a plan signup based on Starter, Professional, and Enterprise.

`dashboard-signup.html` `signup.html` plan flow Route separation

Signup Polish

Signup pages now share the same site chrome

We finished by aligning the signup pages with the main site shell, switching the dashboard signup wording from persona language to user language, and reusing the same dashboard footer so the signup experience feels consistent end to end.

Shared signup header Dashboard footer reused User wording
